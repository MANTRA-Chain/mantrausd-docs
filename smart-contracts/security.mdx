## Best Practices for Users

### Verify Contract Address

Always verify you're interacting with the correct contract:

```
0xd2b95283011E47257917770D28Bb3EE44c849f6F
```

### Check Transaction Details

Before confirming any transaction:
- Verify the contract address
- Check the amount being sent
- Review gas fees
- Confirm the recipient address

### Use Official Channels

- Only use official documentation and links
- Verify information through multiple sources
- Be cautious of phishing attempts

<Warning>
  Never share your private keys or seed phrases with anyone. The mantraUSD team will never ask for this information.
</Warning>

## Reporting Security Issues

If you discover a security vulnerability:

1. **Do not** disclose it publicly
2. Contact the security team through official channels
3. Provide detailed information about the vulnerability
4. Allow time for the issue to be addressed
